# PokedexAPI

Welcome to the PokedexAPI project! This application fetches and displays Pokémon data from the PokeAPI. You can view Pokémon details and search for specific Pokémon by name.

- [Installation](#installation)
- [Usage](#usage)
- [Features](#features)

## Installation

### Prerequisites

- .NET SDK
- Visual Studio or any other C# IDE

### Steps

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/pokedex-api.git
    ```

2. Open the project in Visual Studio:
    - Open Visual Studio.
    - Go to `File > Open > Project/Solution`.
    - Navigate to the cloned repository and open `PokedexAPI.sln`.

3. Build the project:
    - Go to `Build > Build Solution` or press `Ctrl+Shift+B`.

## Usage

1. Start the application:
    - Press `F5` or go to `Debug > Start Debugging`.

2. Load Pokémon data:
    - Click the "Load Pokemons" button to fetch and display Pokémon data from the PokeAPI.

3. Search Pokémon:
    - Enter the name of the Pokémon in the search box and click the "Search" button to filter the displayed Pokémon list.

## Features

- **Load Pokémon Data**: Fetches Pokémon data from the PokeAPI and displays it in a list.
- **Search Pokémon**: Allows searching for Pokémon by name.
- **Type Coloring**: Pokémon types are color-coded for easy identification.
