# Snake Game

Welcome to the Snake Game project! This project is a simple implementation of the classic Snake game using WPF (Windows Presentation Foundation) in C#. The game features basic functionalities like moving the snake, eating food, growing in size, and game-over detection.

## Installation

### Prerequisites

- .NET SDK
- Visual Studio or any other C# IDE

### Steps

1. Clone the repository:
    ```sh
    git clone https://github.com/yourusername/snake-game.git
    ```

2. Open the project in Visual Studio:
    - Open Visual Studio.
    - Go to `File > Open > Project/Solution`.
    - Navigate to the cloned repository and open `SnakeGame.sln`.

3. Build the project:
    - Go to `Build > Build Solution` or press `Ctrl+Shift+B`.

## Usage

1. Start the game:
    - Press `F5` or go to `Debug > Start Debugging`.

2. Control the snake using the arrow keys on your keyboard.

3. Try to eat the food (red squares) to grow in size and increase your score.

4. If the snake runs into itself or the walls, the game will be over.

5. Click on the "Restart the Game" button to play again.

## Features

- **Game Grid**: A 15x15 grid where the snake can move.
- **Score Display**: Displays the current score.
- **Snake Movement**: Control the snake using arrow keys.
- **Food**: Food appears at random positions on the grid.
- **Game Over Overlay**: Displays a "GAME OVER" message when the game ends.
- **Restart Button**: Allows you to restart the game after it ends.
